# Hibernate CRUD Project
A simple Java project demonstrating CRUD (Create, Read, Update, Delete) operations using Hibernate ORM with MySQL database.
This project was built after implementing CRUD using JDBC to understand how Hibernate simplifies database interaction through Object Relational Mapping (ORM).

# Features
Insert student records
Fetch student records
Update student records
Delete student records
Automatic table creation
Entity mapping using Hibernate annotations

# Project Structure
Student – Entity class mapped to the database table
HibernateUtil – Hibernate configuration and SessionFactory setup
InsertStudent – Class for inserting student records
FetchStudents – Class for fetching student records
UpdateStudent – Class for updating student details
DeleteStudent – Class for deleting student records

# What I Learned
Hibernate configuration using hibernate.cfg.xml
Mapping Java classes to database tables using annotations
Using Session and Transaction for database operations
Automatic schema generation from entity classes
Understanding deprecated Hibernate methods
Difference between JDBC and Hibernate
